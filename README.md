# Ostromgep

**Ostromgep** is an Android workout application designed for efficiency and results. It focuses on a streamlined, dark-themed experience with advanced tracking and AI-driven planning.

## Key Features

- **AI Workout Generator**: Powered by **Gemini 2.5 Flash**, the app generates personalized workout routines based on your available equipment, training frequency, custom goals or injuries.
- **Advanced Tracking**: Record sets, reps, and weight with ease. Support for **Supersets**.
- **Smart Rest Timer**: Foreground service-based rest timer with live notifications, keeping you on track between sets.
- **Progress Visualization**: Track your training volume over time with intuitive charts.
- **Exercise Library**: Built-in video demonstrations for correct form guidance.
- **Fast Search**: Optimized fuzzy-matching search for quickly finding exercises by name or muscle group.

## Design
- **Dark Mode Only**: A sleek, high-contrast dark interface optimized for gym environments and focus.
- **Custom Theming**: Pick the color you like the most.

## Tech Stack
- **Jetpack Compose**: For building the UI.
- **Gemini 3.0 Flash**: For AI-powered workout generation.
- **Media3 ExoPlayer**: For video playback.
- **Room**: For local data storage.
- **Hilt**: For dependency injection.
- **Kotlin Flow**: For reactive programming.
- **Kotlin Coroutines**: For asynchronous programming.
- **Jetpack Navigation**: For navigation between screens.
- **Jetpack WorkManager**: For background tasks.
- **Jetpack Compose Charts**: For data visualization.
- **FuzzySearch**: For fast search.
